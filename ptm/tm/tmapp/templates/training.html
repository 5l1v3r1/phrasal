Experiment Goals: Translation Speed & Quality

During the study, translate the documents as quickly and as accurately as you can.

Speed: Once you start working on a document, you may not idle for longer than 3 minutes within a single session. If you pause---meaning an absence of keyboard or mouse input---for 3 continuous minutes, your session will be automatically terminated. The payment for the equivalent number of words will be deducted from your final pay. Continuing to translate will reset the “idle timer.” Otherwise, you will not be penalized or rewarded for total amount of time you spend on a translation.

Quality: Once you submit a translated document, you will not have another opportunity to edit it. Make sure that your translation is in good shape before you press the submit button!

Overview of the Experiment

This website will administer the translation job, which is organized into two modules separated by a break:

  • Translation Module #1 (about 1,200 source words)
  • Break (not timed)
  • Translation Module #2 (about 1,200 source words)
  • Exit Survey

We ask that you complete each translation module in a continuous session. Once you complete Module #1, the website will prompt you to take a break, and the idle timer will be disabled. Get up, go to the bathroom, get some coffee: whatever you need to stay productive. We expect that you'll take about 15 minutes, and no more than an hour. We require that you complete both modules on the same day.

{% elif step == 4 %}

Job Description and Requirements

You will be asked to translate two sets of documents from {{ src_lang }} to {{ tgt_lang }}.

The document may be unrelated to each other. You will be presented with paragraphs selected from the documents. You will see the sentences from these paragraphs, in the same order in which they appear in the original document.

This text comes from technical domains, so it will challenge you. Please make sure that your {{ tgt_lang }} translation:

  • Is faithful to the original text in both meaning and style
  • Is grammatical, fluent, and natural-sounding
  • Does not add or delete information from the original text
  • Does not contain any spelling errors

When creating your translation, you:

  • May use dictionaries, either offline or online (e.g., WordReference).

However:

  • Do not use any other automatic/machine translation systems (e.g., Google Translate, Trados, etc.)
  • Do not use translation memories or term bases
  • Do not use browser plugins or extensions

We will check the translations you submit against those produced by various machine translation systems. We're trying to understand how this specific CAT system works, so please help us by not using other machine assistance.

The two translation modules have different interfaces.

Translation Interface A (Post-edit)

You will be presented with {{ src_lang }} documents in a sequence. The interface will contain textboxes to enter {{ tgt_lang }} translations. Each textbox will be pre-populated with a suggested {{ tgt_lang }} translation. You may accept, edit, or delete the suggested translation. Please work as quickly and as accurately as you can. The "idle timer," which limits your pauses to 3 minutes, is located at the bottom of your browser window. If you do not complete the translation before the idle timer expires, then the incomplete translation will be submitted for you. The site will then redirect you to the next document.

Translation Interface B (Interactive)

You will be presented with {{ src_lang }} documents in a sequence. The interface will provide several interactive aids to help you translate each document into {{ tgt_lang }}. The objective remains the same: translate each document as quickly and as accurately as you can. As before, the "idle timer" will appear at the bottom of your browser window.
